# clock_dear_python
Pythonで作ったデスクトップアプリです。時計アプリです。exeなんて糞食らえ。

## 使用したライブラリ

主なライブラリを紹介

| 使用ライブラリ | 説明 |
----|---- 
| tkinter | GUIを構築・操作 |
| datetime | 日付・時間を取得 |
| pydub | mp3ファイルの操作 |
| pygame | 今回はmp3ファイルの再生に使用 |

## ファイル構成

root/<br>
　├ clock.py<br>
　├ time/<br>
　│　└ mp3ファイル<br>
　└ nohup.out<br>
 
※ nohup.outは不慮の事故で出てきます。一応.gitignoreで弾いてます

### timeフォルダ内のmp3ファイルについて

時間を発声するファイルに関しては、「time_{時間}.mp3」って感じにしている。
